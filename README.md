Este projeto recria a mecânica do famoso Flappy Bird, onde o jogador controla um pássaro que deve desviar de obstáculos (canos) para acumular pontos.

O objetivo é testar habilidades de lógica, física simples e manipulação gráfica em tempo real.

Funcionalidades:

Controle do personagem com a tecla ESPAÇO
Geração dinâmica de obstáculos (canos)
Sistema de pontuação em tempo real
Detecção de colisão precisa
Animação de fundo e chão em movimento
Loop contínuo de jogo

Tecnologias utilizadas:
Python 3
Pygame
